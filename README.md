# 移动端基础开发模板

## 安装依赖
```
npm install
```

### 本地开发
```
npm run serve
```

### 测试发包
```
npm run test
```

### 生产发包
```
npm run build
```

### 本地预览
```
npm run view
```

### node版本
```shell
node -v 14.15.4
```

### vue-cli版本
```shell
@vue/cli 4.5.19
```
